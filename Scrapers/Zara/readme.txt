Should have install python 3+

Should have install firefox web browser version 95.0.2 install

firefox webdriver for mac should be in firefox_webdriver folder 

create a virtual environment by this command "python3 -m venv ./venv

activate the virtual environment using this command "source ./venv/bin/activate"

Install the required libraries given in "requirements.txt" file by the following command
                               "pip3 install -r requirements.txt"

Run the following command to run the script 
                               "python3 zara.py"

In Data folder "output.json" will be generated and all scraped data will append to csv file 

This code is tested on mac python version python 3.8.9 screenshot is also added of running commands.
